# Student Exam Score Prediction using Machine Learning

## 📖 Description

This project predicts student exam scores based on study habits and personal factors using a Linear Regression model. It includes data preprocessing, encoding, model training, and evaluation.

## 🚀 Features

* Data preprocessing and cleaning
* Label Encoding and One-Hot Encoding
* Model training using Linear Regression
* User input-based prediction
* Model evaluation using MAE, MSE, and R² Score

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python your_file_name.py

## 📊 Model Information

* Algorithm: Linear Regression
* Evaluation Metrics:

  * MAE (Mean Absolute Error)
  * MSE (Mean Squared Error)
  * R² Score

## 📌 Input Features

* Study Hours
* Study Method (Self/Coaching)
* Internet Access (Yes/No)
* Gender (Male/Female)
